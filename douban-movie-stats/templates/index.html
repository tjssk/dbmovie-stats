工具说明

1. 采集流程：通过 uid 翻页获取你的"看过"列表 → 调用豆瓣 API 获取每部电影的导演、类型、国家、上映年份等信息 → 从列表页信息行提取语言。

2. 统计维度：导演出现次数、影片类型分布、制片国家/地区分布、语言分布、上映年份/年代分布、你的评分分布、豆瓣评分分布、观影时间线。

3. 缓存机制：已采集的电影详情会缓存到本地，下次运行时自动跳过已缓存的影片，仅采集新增的观影记录。勾选"强制刷新"可重新采集全部。

4. 反爬说明：为避免被豆瓣限制，每次请求间隔约 2 秒。200 部电影约需 7-8 分钟，请耐心等待。采集过程中可随时关闭页面，下次运行会从断点继续。

5. 数据用途：统计个人观影偏好 / 辅助论文选题 / 发现被忽略的影片范围，查漏补缺。
